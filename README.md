# WebAppRentAcar
WebApp Rent A Car

Описание на проекта
WebApp Rent A Car е уеб приложение, разработено с ASP.NET Core MVC, което позволява управление на автомобили и резервации. Потребителите могат да разглеждат наличните автомобили и да създават резервации за определен период, а администраторите имат възможност да одобряват тези резервации.

Основна цел
Целта на проекта е да се създаде система за управление на автомобили и резервации с използване на модерни технологии и добра архитектура.

Използвани технологии
ASP.NET Core MVC
Entity Framework Core
Microsoft SQL Server
ASP.NET Core Identity
Bootstrap
LINQ

Сигурност
Използва се ASP.NET Core Identity за автентикация и авторизация. Реализирана е ролева система с потребители и администратори, като административните функции са ограничени само за администратори.

База данни
Основните таблици са Users, Cars и Reservations. Един потребител може да има много резервации, а един автомобил може да бъде резервиран многократно за различни периоди.

Стартиране на проекта
Отворете проекта с Visual Studio 2022. Изпълнете миграциите чрез Add-Migration и Update-Database. Стартирайте приложението чрез IIS Express.
